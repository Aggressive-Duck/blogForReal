## Project Goal

To create a personal blog that balances a **minimalist visual aesthetic** with **technically interesting** implementation details.

## Core Structure

- **Home Page:** Simple, curated selection of links/topics. No "fluff" or excessive decorative elements.
- **Blog Area:** The primary focus of the application. Designed for a high-quality reading experience.

## Features

- **Admin/Editor Area:** A secured section to write blog content directly within the site.
- **Live Preview:** A feature to view how content looks before publishing.

## Tech Stack

- **Frontend Framework:** React Router (v7)
- **Styling:** Tailwind CSS (v4)
- **Backend API:** Node.js with Express
- **Database:** PostgreSQL
